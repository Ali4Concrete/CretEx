# CretEx: Concrete Testing Excellence

### The Digital Compliance System for Civil Engineering Labs

**CretEx** is an open-source Python ecosystem designed to transform static ASTM/ACI standards into executable, type-safe, and physics-informed code. It aims to eliminate calculation uncertainty in ISO 17025 accredited laboratories.
